# Layouts

Use these names in `outline.md` when helpful.

| Layout | Use |
|--------|-----|
| `title-hero` | cover slide with large title and one strong visual |
| `section-divider` | new chapter or pivot point |
| `two-column` | concept plus explanation, before/after |
| `three-cards` | three related ideas |
| `timeline` | sequence of events |
| `process-flow` | steps or pipeline |
| `hub-spoke` | one central idea with implications |
| `comparison` | tradeoffs or alternatives |
| `data-callout` | one important number with context |
| `matrix` | two-axis categorization |
| `quote-focus` | one quote with interpretation |
| `closing-card` | final takeaway or next action |

Layout guidance is descriptive. Do not force a layout when the slide content suggests a better arrangement.
